LED matrix control

colors
Predefined pixel colors.

Fields:

  • red
  • green
  • blue
  • yellow
  • violett
  • white
  • black
pattern
LED matrix Patterns.

Fields:

  • off display off
  • on display on
  • checkerboard checkerboard
getLetterMap (s)
Get the letter map for a letter.

Parameters:

  • s string to look up.

Returns:

    pixel field representing the letter.
packPixel (pix)
Convert pixel into 16 bit rgb565 value. Format: (hi,lo) = (rrrrrggg, gggbbbbb).

Parameters:

  • pix Pixel {r,g,b} to convert.

Returns:

    Two byte value representing pixel in rgb565 notation.
unpackPixel (rgb)
Convert rgb565 value into pixel.

Parameters:

  • rgb Number representing pixel to convert.

Returns:

    Table {r,g,b} describing the pixel.
getRotation ()
Get current rotation of diaplay.

Returns:

    Rotation as number 0, 90, 180 or 270.
setPixel (x, y, ...)
Set a single pixel.

Parameters:

  • x Number representing x coordinate.
  • y Number reprenseting y coordinate.
  • ... PIxel as table or numbers.

Usage:

  • setPixel(1, 2, r, g, b)
  • setPixel(1, 2, {r, g, b})
getPixel (x, y)
Get pixel from frame buffer at given position.

Parameters:

  • x X coordinate.
  • y y coordinate.

Returns:

    Pixel as table {r,g,b}.
setPixels (pixels)
Display pixels defined in a list of pixels. The function may display only a subset of possible 64 (8x8) pixels.

Parameters:

  • pixels List of pixels {{r,g,b}, ..., {r,g,b}}
getPixels ()
Read pixels from frame buffer. The function always returns all 64 (8x8) pixels.

Returns:

    List of pixels.
setRotation (r, redraw)
Set Rotation.

Parameters:

  • r Rotation value: 0, 90,180, 270 degree.
  • redraw Boolean value. If set the display is redrawn. If not set, only the module's rotation parameter is set.
clear (...)
Clear the display. If no color is given 'black' is used as color.

Parameters:

  • ... Table with color definition.

Usage:

  • clear(r,g,b)
  • clear{r,g,b}
showLetter (s, fg, bg)
Show a letter.

Parameters:

  • s Letter to display (string).
  • fg Foreground color as table (default: white).
  • bg Background color as table (default: black).
loadImage (file, redraw)
Load an image from file into a pixel list and optinally display. Note: We use image magick convert utility to read the the pixels of the image directly into a Lua string via pipe.

Parameters:

  • file File to load and optionally display.
  • redraw Boolean value. If true, show image on display.

Returns:

    List of pixels.
flipHorizontal (redraw)
Flip display horizontal.

Parameters:

  • redraw Redraw if nil or true (default true).

Returns:

    Flipped Pixels
flipVertical (redraw)
Flip display vertical.

Parameters:

  • redraw Redraw if nil or true (default true).

Returns:

    Flipped Pixels
showMessage (message, speed, fg, bg)
Display a moving message.

Parameters:

  • message Message to display.
  • speed Speed in seconds per pixel shift.
  • fg Foreground color.
  • bg Background color.
setGamma (buf)
The gamma attributes provides a means to define the least significant 5 bits of a color value. These bits are replaced by the value given in a lookup table using the index as key and the stored value as replacement. The value must be in the range of 0 to 31.

Parameters:

  • buf Lookup table with 32 values.
getGamma ()
Retviews the current active gamma lookup table.

Returns:

    Current active gamma lookup table.
resetGamma ()
Reset gamma lookup table to default.
getLowLight ()
Determine whether gamma is adusted to low light.

Returns:

    true or false.
setLowLight (value)
Set gamma to low light

Parameters:

  • value true or false.

Turn LED matrix off.
ledOn ()
Turn LED matrix on.
